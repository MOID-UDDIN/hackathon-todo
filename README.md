# Hackathon Todo App

A full-stack Todo application built for Hackathon II using a spec-driven development approach.

## Overview
This project consists of a FastAPI backend and a Next.js frontend. The backend handles authentication, todos, and chatbot logic, while the frontend provides a simple user interface for managing todos and interacting with the chatbot.

## Tech Stack
- Backend: FastAPI, SQLModel
- Frontend: Next.js (React, TypeScript)
- Database: PostgreSQL (Neon)
- Authentication: JWT
